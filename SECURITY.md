# NOVAX Admin - سياسات الأمان (Security Policy)

> **آخر تحديث:** 2025-12-19

---

## 1. سياسة عدم الأسرار (No-Secrets Policy)

### ممنوع منعاً باتاً:
- ❌ رفع ملفات `.env` أو `.env.local` الحقيقية
- ❌ كتابة API keys أو tokens في الكود
- ❌ رفع أي credentials

### مسموح:
- ✅ ملف `.env.example` بقيم وهمية
- ✅ الأسرار في Vercel Environment Variables
- ✅ الأسرار في GitHub Secrets

---

## 2. نموذج الوصول (Access Model)

لوحة الإدارة تتطلب:
- تسجيل دخول عبر API Backend
- JWT token صالح
- صلاحية `admin` أو أعلى
